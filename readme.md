# T20 World Cup Live Dashboard

A live cricket dashboard that shows:
- Team scoreboard (runs, wickets, overs)
- Player fantasy-style points
- Hat-trick detection
- Auto-refresh every 30 seconds

## Setup
1. Get a free API key from [CricAPI](https://www.cricapi.com).
2. Replace `YOUR_API_KEY` in `index.html` with your key.
3. Host the project:

### GitHub Pages
- Create a new repository.
- Upload `index.html` and `README.md`.
- Enable GitHub Pages in repo settings.
- Your site will be live at `https://username.github.io/t20-dashboard`.

### Netlify
- Go to [Netlify Drop](https://app.netlify.com/drop).
- Drag and drop the folder.
- Netlify gives you a live link instantly.

## Notes
- Refreshes every 30 seconds for live updates.
- Works best during live T20 World Cup matches.
